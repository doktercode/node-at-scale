Node.js Core Modules:
=====================

    * assert    -   used by node.js for testing
    * buffer    -   perform operations of raw bytes of data which reside in memory.
    * child_process -  used by node.js for managing child_process.
    * cluster   -   to take advantage of multi-core systems
    * console   -   write data to console
    * crypto    -   support cryptography for encryption and decryption
    * debugger  -   used for code debugging.
    * dns       -   ops like lookup and resolve domain names
    * events    -   events handling in nodejs, events are emitted by other node objects.
    * fs        -   ops on files.
    * http      -   create http server and client.
    * net       -   tcp server and client which can communicate using TCP protocol
    * os        -   OS level utility functions
    * path      -   ops on path of files
    * process   -   global objects and provides info about program execution.
    * qs        -   deal with query strings
    * stream    -   stream data between 2 entities
    * timers    -   All timers functions are global and deal with time.
    * url       -   url resolution and parsing
    * util      -   designed to support node.js internal API's 
    * vm        -   provides access to v8's VM to compile and execute JS code.
    * zlib      -   compress and decompress data.
